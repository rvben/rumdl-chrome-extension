## Short Description (132 chars max)

Real-time Markdown linting for GitHub & GitLab. Catch issues as you type with inline warnings and one-click auto-fix.

## Detailed Description

rumdl brings real-time Markdown linting directly to your browser. Get instant feedback as you write issues, pull requests, comments, and wiki pages on GitHub and GitLab.

**Features:**
- Real-time linting as you type with 60+ rules
- Gutter dots showing exactly which lines have issues
- Warning panel with full issue details
- One-click "Fix All" for auto-fixable issues
- Keyboard shortcuts for fast navigation
- Configurable rules via popup settings
- Light and dark theme support
- Zero network requests — all processing happens locally via WASM

**Supported sites:**
- GitHub (issues, PRs, comments, wikis, discussions)
- GitLab (issues, merge requests, comments, wikis)

**Keyboard shortcuts:**
- Cmd/Ctrl+Shift+L — Toggle warning panel
- Cmd/Ctrl+Shift+F — Fix all issues
- Cmd/Ctrl+. — Fix issue at cursor
- Cmd/Ctrl+Alt+] — Next warning
- Cmd/Ctrl+Alt+[ — Previous warning

Powered by rumdl, a fast Markdown linter written in Rust and compiled to WebAssembly. Open source at github.com/rvben/rumdl.
